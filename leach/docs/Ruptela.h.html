Ruptela.h

Purpose

Declares the Ruptela service simulation types, constants, and public API. This header is only active when RUPTELA_SERVICE_SIMULATION is enabled.

Data types

  • Ruptela_Log: timestamped log record with codes and values.
  • Ruptela_Command: parsed command payload structure.

Log code constants

Defines log codes such as RUPTELA_LOG_DISARMED, RUPTELA_LOG_PASSWORD_CHANGED, and RUPTELA_LOG_LEACH_CONNECTED. Codes are encoded as 0xAAAABBBB where AAAA is the log code and BBBB is the sub-code.

Command code constants

Defines commands like RUPTELA_COMMAND_DISARM, RUPTELA_COMMAND_NEW_PASSWORD, and RUPTELA_COMMAND_SET_TIME. Codes are encoded as 0xAAAABBBB.

Public API

  • read_new_security_state
  • ruptela_send_log
  • ruptela_decode_command
  • getEnablePairing
  • resetEnablePairing

Notes

  • Assumes SecurityState is defined elsewhere.
  • Declarations are guarded by RUPTELA_SERVICE_SIMULATION.
