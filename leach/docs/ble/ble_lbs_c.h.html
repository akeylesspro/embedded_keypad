ble_lbs_c.h

Purpose

Public API and type definitions for the LED Button Service (LBS) client.

UUID and characteristic IDs

  • LBS_UUID_BASE and LBS_UUID_SERVICE
  • LBS_UUID_BUTTON_CHAR, LBS_UUID_LED_CHAR
  • Simulation-only: LBS_UUID_COMMAND_CHAR, LBS_UUID_LOG_CHAR

Key types

  • ble_lbs_c_t: client instance state.
  • lbs_db_t: peer handles for characteristics.
  • ble_lbs_c_evt_t: events for discovery and notifications.

Public API

  • ble_lbs_c_init
  • ble_lbs_c_on_ble_evt
  • ble_lbs_on_db_disc_evt
  • ble_lbs_c_button_notif_enable
  • ble_lbs_c_handles_assign
  • ble_lbs_led_status_send
  • ble_lbs_log_send

Notes

Simulation-specific fields are guarded by RUPTELA_SERVICE_SIMULATION.
