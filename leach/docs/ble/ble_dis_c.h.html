ble_dis_c.h

Purpose

Public API and types for the Device Information Service (DIS) client.

Key types

  • ble_dis_c_t: client instance state.
  • ble_dis_c_evt_t: event container for discovery and read responses.
  • ble_dis_c_char_type_t: supported DIS characteristic types.
  • ble_dis_c_init_t: initialization parameters.

Macros

  • BLE_DIS_C_DEF and BLE_DIS_C_ARRAY_DEF

Public API

  • ble_dis_c_init
  • ble_dis_c_on_db_disc_evt
  • ble_dis_c_on_ble_evt
  • ble_dis_c_read
  • ble_dis_c_handles_assign

Notes

Some DIS characteristic types are disabled via comments.
