nrf_ble_scan.c

Purpose

Implements the Nordic BLE scanning module. It provides scanning, filtering, and optional auto-connect behavior.

Main responsibilities

  • Initialize scanning and connection parameters.
  • Maintain scan filters (name, address, UUID, appearance, short name).
  • Parse advertisement reports and determine filter matches.
  • Auto-connect when a filter matches and auto-connect is enabled.
  • Raise scan events to the application.

Key behaviors

  • nrf_ble_scan_init configures defaults or provided params.
  • nrf_ble_scan_start / nrf_ble_scan_stop
  • nrf_ble_scan_filter_set and filter enable/disable APIs.
  • nrf_ble_scan_on_ble_evt handles BLE events.

Filter logic

  • Supports multi-filter match and match-all mode.
  • Address filtering can compare against target lists.
  • UUID filtering can be combined with MAC filtering.

Notes

Contains project-specific MAC address filter integration.
