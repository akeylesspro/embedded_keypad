ble_services_manager.h

Purpose

Public interface for BLE services management.

Public API

  • Initialization: ble_services_init_0, ble_services_init_peripheral, ble_services_init_central
  • Advertising: ble_services_advertising_start
  • Scanning: scan_start, scan_stop, scan_init_by_white_list, scan_init_by_uuid
  • Status: ble_services_is_connected
  • Utility: decode_peripheral_device_by_dis

Data structures

ble_peripheral_info_t tracks ignition state, connection time, peer address, connection handle, and peripheral type.

Dependencies

Relies on BLE GAP and DIS types.

Notes

ble_services_init_peripheral is declared but implemented elsewhere.
