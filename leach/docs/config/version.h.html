version.h

Purpose

Defines firmware identity information used in BLE advertising and DIS characteristics.

Key macros

  • DEVICE_NAME
  • MANUFACTURER_NAME
  • MODEL_NUM and HARDWARE_REV
  • FIRMWARE_REV

Version history

Contains a detailed commented history of firmware revisions. The active revision is the single uncommented FIRMWARE_REV.

Notes

  • Board selection affects model and hardware revision strings.
  • version[] is declared externally for runtime use.
