FreeRTOSConfig.h

Purpose

FreeRTOS kernel configuration for the Leach firmware. It defines scheduler behavior, memory sizes, interrupt priorities, and features.

Key scheduler settings

  • Preemption enabled and tickless idle enabled.
  • Tick source uses RTC with a 1024 Hz tick rate.

Memory and task settings

  • Heap size configured via configTOTAL_HEAP_SIZE.
  • Minimal stack size and timer task stack/queue sizes defined.

Synchronization features

Mutexes, recursive mutexes, and counting semaphores are enabled.

Interrupt configuration

  • Kernel and syscall interrupt priorities are explicitly set.
  • RTC1 is used as the tick interrupt.

Hooks and diagnostics

  • Idle hook enabled.
  • Stack overflow and malloc failed hooks disabled.
  • Optional debug asserts via configASSERT.

Notes

Settings are tuned for low-power embedded use.
