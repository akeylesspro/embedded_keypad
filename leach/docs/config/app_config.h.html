app_config.h

Purpose

Project-specific configuration overrides for the Leach firmware. It layers on top of shared configuration and defines BLE and scanning parameters.

Key configuration areas

  • BLE observer priority and log level.
  • Central/peripheral link counts.
  • GATT Queue enablement and sizing.
  • Scanning parameters and filter capacities.

Notable macros

  • Link counts: NRF_SDH_BLE_CENTRAL_LINK_COUNT, NRF_SDH_BLE_PERIPHERAL_LINK_COUNT
  • GATT Queue sizing: NRF_BLE_GQ_*
  • Scan params: NRF_BLE_SCAN_*
  • RUPTELA_SERVICE_SIMULATION
  • MAX_TARGET_ADDRESSES

Notes

Uses shared_app_config.h as a base and supports Nordic config tools.
